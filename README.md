# Геолого-технологическая классификация руд

## О проекте

End-to-end система автоматической классификации медно-никелевых руд по панорамным OM-изображениям полированных шлифов.

**Команда:** VlaMMa

## Ключевые возможности

- **Сегментация сульфидных включений** с классификацией на обычные и замещённые срастания
- **Детекция талька** — тёмной рассеянной фазы в нерудной матрице
- **Экспертная классификация** по геолого-технологическим сортам: рядовая, труднообогатимая, оталькованная
- **Устойчивость к вариациям данных** — работает на снимках с разным освещением, контрастом и цветовой палитрой
- **Интерпретируемость** — цветовая маска, таблица метрик и текстовое заключение
- **Контейнеризация** — готовность к развёртыванию в лабораторном пайплайне

## Требования

### Локальный запуск
- Python 3.11+
- RAM: 4+ ГБ
- CPU: 2+ ядра

### Docker
- Docker Engine 20.10+
- Docker Compose 2.0+

## Установка и запуск

### Вариант 1: Локально

```bash
# Клонирование репозитория
git clone <repository-url>
cd ore_classifier

# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
streamlit run ore_analyzer.py
```

Приложение откроется по адресу: `http://localhost:8501`

### Вариант 2: Docker

```bash
# Сборка образа
docker build -t ore-classifier:latest .

# Запуск контейнера
docker run -d \
  --name ore-classifier \
  -p 8501:8501 \
  -v $(pwd)/data:/app/data \
  -v $(pwd)/results:/app/results \
  ore-classifier:latest

# Или через Docker Compose
docker-compose up -d
```

Проверка работоспособности:
```bash
curl http://localhost:8501/_stcore/health
```

## Использование

1. **Загрузите панораму** через боковую панель (поддерживаются TIFF, PNG, JPEG)
2. **Настройте параметры** при необходимости:
   - Пороги классификации (тальк, замещение)
   - Параметры детекции (перцентили, морфология)
   - Метод нормализации (adaptive / CLAHE / histogram)
   - Метод детекции талька (CV / UNet)
3. **Запустите анализ** — система выполнит сегментацию и классификацию
4. **Изучите результаты:**
   - Цветовая маска (зелёный — обычные, красный — замещённые, синий — тальк)
   - Таблица количественных метрик
   - Текстовое заключение
5. **Экспортируйте отчёт** в CSV и PNG
